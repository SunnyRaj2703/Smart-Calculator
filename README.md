# Smart-Calculator
A modern Smart Calculator built using HTML, CSS, and JavaScript.
It supports real-time calculation, keyboard input, delete and clear functions, and includes a full calculation history with show/hide options. Clean UI, smooth animations, and fully responsive design.
# Features
Supports addition, subtraction, multiplication, and division

Real-time display updates

History panel with show/hide feature

Delete (DEL) & Clear (C / AC) functions

Keyboard support (0-9, operators, Enter, Backspace)

Fully responsive and clean UI

Smooth animations and interactive buttons

# How It Run
Click buttons or use keyboard to enter numbers

Press = or Enter to calculate

Results are added to history automatically

Click Show History to toggle history panel

Click AC to clear history
